# ExploreKosova
ExploreKosova është një webfaqe moderne front-end e ndërtuar me HTML, CSS dhe JavaScript. Projekti paraqet një agjenci udhëtimesh në Kosovë me faqe të strukturuara, dizajn minimal, imazhe profesionale dhe forma të validuara me JavaScript. Zhvilluar si projekt semestral për lëndën Dizajn dhe Zhvillim i Ueb-it.
